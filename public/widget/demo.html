Embeddable Chat Widget — Local Demo

This page loads the widget for local testing. The chat bubble appears in the bottom-right corner.

How to use this page

  1. Make sure the app is running on http://localhost:3000 (e.g. npm run dev).
  2. Replace PUBLIC_BOT_ID in the script tag below with a real bot's public id. You can copy it from the company dashboard → Assistant Settings → Embed snippet.
  3. Open this file through the app (http://localhost:3000/widget/demo.html) so the widget script and the API share the same origin.

The embed snippet looks like this:

<script
  src="http://localhost:3000/widget/widget.js"
  data-bot-id="PUBLIC_BOT_ID"
  data-title="Acme Assistant"
  data-welcome="Hi! Ask me anything.">
</script>

Optional attributes: data-api, data-color, data-position="left|right", data-lang="en|ar|auto".
